A directory of community-built external tools for ImageGlass. ImageGlass allows you to register external applications to seamlessly extend its capabilities.

**Legends**:
- Supported platforms: 🪟 Windows, 🍎 macOS, 🐧 Linux
- Use SDK: Tools integrated with [ImageGlass.SDK](https://github.com/ImageGlass/SDK) for inter-process communication (IPC) with the main app.

## FAQs
### 1. How to install a tool
1. Open ImageGlass Settings > Tools.
2. Click the **Add** button to open the **Add external tool** window.
3. Fill in the required information (ID, Name, Executable, and Argument).
   - Tick the option **Integrated with ImageGlass.SDK** if the tool's **Use SDK** is checked.
4. Click **OK**.

### 2. How to submit a tool
1. Fork this repository https://github.com/ImageGlass/SDK.
2. Update this file ('showcase/tools.md') with the tool details in the table.
3. Submit a pull request.
